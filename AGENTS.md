# Project Operating Instructions

This repository is a Phase 1 research prototype for explainable regulatory health informatics in Thai FDA food manufacturing floor-plan pre-screening.

Follow these rules strictly:

- Treat all outputs as advisory decision support only.
- Do not implement automatic approval or rejection of applications.
- Make clear that human Thai FDA officers remain responsible for regulatory decisions.
- Use only synthetic sample data in this repository.
- Do not add real Thai FDA data, applicant names, addresses, license numbers, signatures, official logos, or confidential business data.
- Keep code, comments, logs, and documentation free of emojis and decorative icons.
- Prefer transparent, explainable rules over opaque automation.
- Keep tests based only on synthetic examples.
- Preserve human-in-the-loop review language in generated reports and user interfaces.
