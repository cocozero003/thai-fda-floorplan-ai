# Synthetic Sample Data

This directory contains only synthetic floor-plan annotations created for testing and demonstration. The examples do not represent a real Thai FDA application, applicant, site, address, license, production line, or confidential business layout.

The sample annotation intentionally includes advisory risk conditions:

- Toilet too close to production area.
- Waste flow crossing product flow.
- Raw material storage too close to finished product storage.

Use this data only for prototype development, demonstrations, and tests.
